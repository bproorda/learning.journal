## Notes on Markdown ##

**Markdown** : A way to style text on the web, GitHub

- Files saved as .md file then converted, symbol operators modify text.
  - uses non-alphanumeric characters, *, #, etc
- Header
        - # This is an <h1> tag
          ## This is an <h2> tag
          ###### This is an <h6> tag
-Bold
